# pill dispenser app

For the course project on 'Robotics and Automation'

This is a medicine dispenser with multi user support and face recognition to dispense to the right user.
This repo is for the android app written in java.



## Features:

-Android app for user management (http communication).

-LCD Display and menu with buttons to cycle through reminders and dismiss them.

-Stepper motor control for dispensing.

-Custom alarm tone selection from the app

-Multiple medicines for one user and custom timings per medicine
